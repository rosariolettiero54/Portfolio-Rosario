# Portfolio Rosario Lettiero

Sito web personale di Rosario Lettiero, Junior Java Developer.

## Descrizione

Questo progetto rappresenta il portfolio professionale di Rosario Lettiero, sviluppato per presentare le sue competenze, esperienze lavorative, progetti e certificazioni nel campo dello sviluppo software.

Il sito è stato costruito utilizzando **Spring Boot** per il backend e **HTML, CSS, JavaScript** per il frontend, offrendo un'esperienza utente moderna e responsiva.

## Tecnologie Utilizzate

### Backend
- **Java 21**
- **Spring Boot 4.0.3**
- **Spring Boot Starter**

### Frontend
- **HTML5**
- **CSS3**
- **JavaScript**
- **Boxicons** (icone)
- **React** (per il progetto Questboard)

### Strumenti
- **Maven** (gestione dipendenze)
- **Git** (controllo versione)

## Come Eseguire il Progetto

### Prerequisiti
- Java Development Kit (JDK) 21 o superiore
- Maven 3.6 o superiore

### Istruzioni

1. **Clona il repository**
   ```bash
   git clone <repository-url>
   cd PORTFOLIO
   ```

2. **Compila il progetto**
   ```bash
   ./mvnw clean install
   ```

3. **Esegui l'applicazione**
   ```bash
   ./mvnw spring-boot:run
   ```

4. **Accedi al sito**
   Apri il browser e visita: `http://localhost:8080`
